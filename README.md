# Что пользователь может делать на веб-приложении
На главной странице расположен католог товаров. Пользователь может нажать на "подробнее" и узнать информацию об товаре, добавить его в корзину. Данные о товаре и пользователе берутся из базы данных.
Есть кнопка профиля для регистрации или входа в аккаунт. На ней форма заполнения: имя, пароль, тип. При добавке пользоватль отправляется в БД. Если выбран супер администратор или администратор, 
то добавить или удалить товар. Данные берутся из БД.


# Как веб-приложение выглядит
## Главная страница.
Список товаров, который состоит из : фото, названия, описания, цены и типа товара.

## Страница регистрации user
а) Супер администратор
б) Администратор(продавец)
в) Пользователь
  Форма с полями : имя, пароль, тип. Для создания аккаунта с привелегией супер администратора требуется заполнить поле нужным паролем.

## Страница корзины (cart)
На странице корзины отображается таблица, где можно увидеть: фото, название, описание, цена, тип, количество товара, изменять кол-во товара. Подсчет итого(цены).

## Страница товара
На странице товара отображается фото, название, описание, цена, тип. Если выбран супер администратор или администратор, 
то удалить товар.
## Страница add_product
Добавить товар, на странице отображается фото, название, описание, цена, тип товара.
